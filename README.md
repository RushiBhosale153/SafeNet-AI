# SafeNet AI - Cybersecurity Web Application

SafeNet AI is a full-stack cybersecurity platform that helps users detect potential security threats.

## Features

🔐 Secure Authentication with Email OTP  
🛡 Phishing Message Detection  
🌐 Website Malware Scanner (VirusTotal API)  
📧 Email Breach Checker (LeakCheck API)  
🤖 AI Cybersecurity Assistant  
📊 Scan History Dashboard  
🌙 Dark Cybersecurity Theme
